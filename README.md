# Next.js Starter Pack with Redux Toolkit 

Official Link
- [Next.js](https://nextjs.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)

## How to use
Download the example and Extract
```
cd next-with-redux-toolkit
```

Install it

```
npm install 
# or
yarn install
```

Run it
```
npm run dev
# or
yarn run dev
```
